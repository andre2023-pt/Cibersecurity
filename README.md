🔗 [Live Threat Map – Radware](https://livethreatmap.radware.com/)
### 🧠 Como interpretar o mapa de ameaças da Radware

O [Live Threat Map](https://livethreatmap.radware.com/) da Radware mostra **ataques cibernéticos em tempo real**, recolhidos através da rede global de honeypots e sensores da empresa. Aqui estão os principais elementos:

| Elemento            | Significado                                                                 |
|---------------------|------------------------------------------------------------------------------|
| 🌐 **Linhas de ataque**   | Mostram o trajeto de ataques, com origem e destino geográfico.              |
| 💥 **Explosões animadas** | Indicam um ataque ativo a ocorrer naquele momento, por tipo e intensidade. |
| 📌 **Localizações**        | Cidades/países com maior tráfego malicioso detetado.                      |
| 🛠️ **Tipos de ataque**     | Pode incluir: DDoS, Exploits, Malware, Injeções, entre outros.             |
| 📊 **Top atacantes**       | Lista os países ou IPs com maior número de ataques originados.            |
| 📈 **Top alvos**           | Mostra os países/servidores mais atacados nas últimas horas.              |

> ⚠️ Nota: Estes dados são representações *quase em tempo real*, mas podem ser sujeitos a atraso ou agregação.

Este tipo de visualização é útil para compreender tendências globais e a origem de ameaças — excelente para profissionais de SOC, equipas de resposta a incidentes e formação em ciberdefesa.
